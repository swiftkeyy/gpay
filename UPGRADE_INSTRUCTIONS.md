## Проверка работы

### 1. Проверьте основные функции:
- `/start` - главное меню должно работать
- `/seller` - регистрация как продавец
- Просмотр товаров - должны показываться лоты от разных продавцов
- `/deals` - список сделок

### 2. Тестовый сценарий:

#### Как покупатель:
1. Пополните баланс (через админку или вручную в БД)
2. Выберите товар
3. Увидите список лотов от разных продавцов
4. Купите товар
5. Если автовыдача - получите код сразу
6. Если ручная - откройте чат с продавцом
7. Подтвердите получение товара

#### Как продавец:
1. `/seller` - создайте магазин
2. Активируйте продавца через админку
3. Добавьте лот
4. Для автовыдачи - добавьте коды товаров
5. Активируйте лот
6. Получите заказ
7. Для ручной доставки - отправьте товар через чат
8. Получите оплату на баланс
9. Выведите средства

## Автоматические задачи

Рекомендуется настроить cron-задачи для:

1. **Автозавершение сделок** (раз в час):
```python
# auto_complete_deals.py
import asyncio
from app.db.session import SessionLocal
from app.services.deal import DealService

async def main():
    async with SessionLocal() as session:
        deal_service = DealService(session)
        count = await deal_service.auto_complete_expired_deals()
        await session.commit()
        print(f"Auto-completed {count} deals")

if __name__ == "__main__":
    asyncio.run(main())
```

2. **Освобождение просроченных резерваций** (раз в 15 минут):
```python
# cleanup_reservations.py
import asyncio
from app.db.session import SessionLocal
from app.services.lot import LotService

async def main():
    async with SessionLocal() as session:
        lot_service = LotService(session)
        count = await lot_service.cleanup_expired_reservations()
        await session.commit()
        print(f"Released {count} reservations")

if __name__ == "__main__":
    asyncio.run(main())
```

Добавьте в crontab:
```bash
0 * * * * cd /path/to/bot && /path/to/venv/bin/python auto_complete_deals.py
*/15 * * * * cd /path/to/bot && /path/to/venv/bin/python cleanup_reservations.py
```

## Возможные проблемы

### Ошибка миграции
Если миграция не применяется:
```bash
# Проверьте текущую версию
alembic current

# Если нужно, откатите на предыдущую версию
alembic downgrade -1

# Попробуйте снова
alembic upgrade head
```

### Импорты не работают
```bash
# Переустановите зависимости
pip install -r requirements.txt --upgrade
```

### Бот не запускается
Проверьте логи:
```bash
tail -f /path/to/logs/bot.log
```
