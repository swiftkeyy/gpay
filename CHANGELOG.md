# Changelog - Обновления бота

## 14.04.2026 - Маркетплейс и исправления

### ✅ Добавлено

#### Маркетплейс функции:
- 🏪 **Стать продавцом** - регистрация магазина через /seller
- 📦 **Управление лотами** - добавление, редактирование, активация/пауза
- 💰 **Баланс продавца** - просмотр баланса и транзакций
- 💸 **Вывод средств** - запрос на вывод заработанных денег
- 📊 **Статистика продавца** - продажи, рейтинг, отзывы
- 💼 **Мои продажи** - список всех сделок продавца

#### Сделки (Deals):
- 📋 **Список сделок** - просмотр покупок и продаж через /deals
- 💬 **Чат в сделке** - общение между покупателем и продавцом
- ✅ **Подтверждение получения** - покупатель подтверждает доставку
- 📦 **Доставка товара** - продавец отправляет данные покупателю
- ⚠️ **Споры** - открытие диспута при проблемах
- 🔄 **Статусы сделок** - отслеживание этапов сделки

#### Главное меню:
- 🏪 Кнопка "Продавать" - быстрый доступ к функциям продавца
- 💼 Кнопка "Мои сделки" - просмотр всех сделок
- 👮 Кнопка "Админка" - только для администраторов

### 🔧 Исправлено

#### Конвертация Telegram Stars:
- **Было**: 1 звезда = 1 рубль
- **Стало**: 1 звезда = 2 рубля (стандартный курс Telegram)
- Настраивается через `TELEGRAM_STARS_PER_RUB=0.5` в .env

#### Фиксация цен:
- Цены теперь фиксируются при создании заказа
- Сумма не меняется после оформления заказа
- Защита от пересчета при ручной оплате

#### CryptoBot API:
- Обновлен URL API (старый pay.crypt.bot больше не работает)
- Улучшена обработка ошибок HTTP 403
- Добавлены правильные заголовки запросов
- Создан файл CRYPTOBOT_SETUP.md с инструкциями

### 🗄️ База данных

#### Автоматическая миграция:
- Миграции теперь применяются автоматически при старте бота
- Добавлено поле `balance` в таблицу `users`
- Созданы 11 новых таблиц для маркетплейса:
  - `sellers` - продавцы
  - `lots` - лоты/товары продавцов
  - `lot_stock_items` - товары в наличии
  - `deals` - сделки
  - `deal_messages` - сообщения в сделках
  - `deal_disputes` - споры
  - `transactions` - транзакции
  - `seller_withdrawals` - выводы средств
  - `seller_reviews` - отзывы о продавцах
  - `favorites` - избранное
  - `notifications` - уведомления

### 📝 Команды

Новые команды:
- `/seller` - управление магазином продавца
- `/deals` - просмотр сделок (покупки и продажи)

### ⚙️ Настройки (.env)

Новые параметры:
```env
# Конвертация звезд (1 рубль = 0.5 звезды)
TELEGRAM_STARS_PER_RUB=0.5

# CryptoBot (исправлен URL)
CRYPTOBOT_API_BASE_URL=https://testnet-pay.crypt.bot/api
CRYPTOBOT_ENABLED=true
```

## Как использовать новые функции

### Для покупателей:
1. Выберите товар в каталоге
2. Если есть несколько продавцов - выберите лучшее предложение
3. Оформите заказ
4. Общайтесь с продавцом в чате сделки
5. Подтвердите получение товара

### Для продавцов:
1. Нажмите "🏪 Продавать" в главном меню
2. Зарегистрируйте магазин (введите название)
3. Дождитесь активации администратором
4. Добавьте лоты с товарами
5. Управляйте продажами через /seller

### Для администраторов:
1. Нажмите "👮 Админка" в главном меню
2. Активируйте продавцов
3. Управляйте спорами
4. Обрабатывайте выводы средств
