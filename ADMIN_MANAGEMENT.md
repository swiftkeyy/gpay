# 🎮 Game Pay - Управление Ролями

## Доступные Роли

### Административные роли (таблица `admins`)

1. **👑 Супер Админ** (`super_admin`)
   - Полный доступ ко всем функциям админ-панели
   - Управление пользователями, играми, категориями, товарами
   - Просмотр аналитики и статистики
   - Управление другими админами

2. **⚙️ Админ** (`admin`)
   - Управление контентом (игры, категории, товары)
   - Модерация отзывов и споров
   - Просмотр статистики
   - Без доступа к управлению админами

3. **🛡️ Модератор** (`moderator`)
   - Модерация отзывов
   - Управление спорами
   - Просмотр жалоб
   - Без доступа к управлению контентом

4. **🔒 Служба Безопасности** (`security`)
   - Блокировка/разблокировка пользователей
   - Просмотр подозрительной активности
   - Управление спорами
   - Без доступа к управлению контентом

### Продавцы (таблица `sellers`)

**🏪 Продавец** (`seller`)
- Создание лотов
- Управление своими товарами
- Обработка заказов
- Просмотр статистики продаж

---

## Использование

### 1. Добавить админа

```bash
python add_admin.py add <telegram_id> <role>
```

**Примеры:**
```bash
# Добавить супер админа
python add_admin.py add 123456789 super_admin

# Добавить модератора
python add_admin.py add 987654321 moderator

# Добавить службу безопасности
python add_admin.py add 555555555 security
```

### 2. Добавить продавца

```bash
python add_admin.py seller <telegram_id>
```

**Пример:**
```bash
python add_admin.py seller 111222333
```

### 3. Удалить админа

```bash
python add_admin.py remove <telegram_id>
```

**Пример:**
```bash
python add_admin.py remove 123456789
```

### 4. Список всех админов

```bash
python add_admin.py list
```

2. Нажмите `/start`
3. Бот покажет ваш ID

## Важные Замечания

⚠️ **Пользователь должен сначала авторизоваться в приложении!**

Перед добавлением роли пользователь должен:
1. Открыть бота Game Pay
2. Нажать кнопку "🛍️ Открыть магазин"
3. Авторизоваться в Mini App

Только после этого можно добавить роль через команду.

## Примеры Использования

### Сценарий 1: Добавить себя как супер админа

```bash
# 1. Узнайте свой telegram_id через @userinfobot
# 2. Откройте Game Pay Mini App и авторизуйтесь
# 3. Выполните команду:
python add_admin.py add YOUR_TELEGRAM_ID super_admin
```

### Сценарий 2: Добавить команду модераторов

```bash
# Модератор 1
python add_admin.py add 111111111 moderator

# Модератор 2
python add_admin.py add 222222222 moderator

# Проверить список
python add_admin.py list
```

### Сценарий 3: Добавить продавца

```bash
# Добавить статус продавца
python add_admin.py seller 333333333

# Продавец теперь может создавать лоты
```

### Сценарий 4: Изменить роль

```bash
# Повысить модератора до админа
python add_admin.py add 111111111 admin

# Понизить админа до модератора
python add_admin.py add 111111111 moderator
```

## Проверка Доступа

После добавления роли:

1. Перезапустите Mini App
2. Откройте профиль
3. Должна появиться кнопка "⚙️ Админ-панель" (для админов)
4. Или кнопка "💼 Продавать" (для продавцов)

## Troubleshooting

### Ошибка: "User not found"

**Решение:** Пользователь должен сначала авторизоваться в Mini App

### Ошибка: "Invalid role"

**Решение:** Проверьте правильность написания роли:
- `super_admin`
- `admin`
- `moderator`
- `security`

### Роль добавлена, но кнопка не появилась

**Решение:**
1. Полностью закройте Mini App
2. Откройте заново
3. Если не помогло - очистите кэш Telegram

## Railway Deployment

Для добавления админа на Railway:

```bash
# 1. Подключитесь к Railway CLI
railway login

# 2. Выберите проект
railway link

# 3. Выполните команду
railway run python add_admin.py add YOUR_TELEGRAM_ID super_admin
```

Или через Railway Dashboard:
1. Откройте проект
2. Перейдите в Settings → Variables
3. Добавьте переменную `ADMIN_TELEGRAM_IDS` со списком ID через запятую
4. Перезапустите сервис
