# 🎮 Game Pay Marketplace

> Полноценный маркетплейс игровых товаров в Telegram, как FunPay и PlayerOK

## 🚀 Быстрый старт

```bash
# 1. Бэкап БД
pg_dump your_db > backup.sql

# 2. Применить миграцию
alembic upgrade head

# 3. Добавить тестовые данные (опционально)
python seed_marketplace.py

# 4. Запустить бота
python -m app.main
```

## ✨ Основные возможности

### Для покупателей:
- 🛍️ Выбор из множества продавцов
- ⚡ Мгновенная автовыдача кодов
- 💬 Чат с продавцом
- 🔒 Защита сделок (эскроу)
- ⭐ Отзывы и рейтинги
- ❤️ Избранные товары

### Для продавцов:
- 🏪 Свой магазин с профилем
- 📦 Управление лотами
- ⚡ Автоматическая доставка
- 💰 Баланс и вывод средств
- 📊 Статистика продаж
- 💬 Чат с покупателями

### Для админов:
- 👥 Управление продавцами
- ⚖️ Разрешение споров
- 💸 Обработка выводов
- 📈 Аналитика и отчеты

## 📋 Новые команды

- `/seller` - меню продавца
- `/deals` - список сделок

## 🏗️ Архитектура

```
Покупатель → Выбирает товар → Видит лоты от продавцов
           ↓
Выбирает лот → Оплачивает → Создается сделка
           ↓
Автовыдача ИЛИ Чат с продавцом
           ↓
Подтверждает получение → Средства продавцу → Завершено
```

## 🎯 Основные функции

### 1. Система продавцов
```
/seller → Создать магазин → Активация админом → Готов продавать
```

### 2. Лоты
```
Продавец → Создает лот → Устанавливает цену → Добавляет товар → Активирует
```

### 3. Автовыдача
```
Продавец загружает коды → Покупатель покупает → Код выдается мгновенно
```

### 4. Сделки
```
Покупка → Создание сделки → Эскроу → Доставка → Подтверждение → Завершение
```

### 5. Чат
```
Покупатель ↔️ Чат ↔️ Продавец
```

## 📦 Что включено

### Модели БД:
- `Seller` - продавцы
- `Lot` - лоты/предложения
- `LotStockItem` - товары для автовыдачи
- `Deal` - сделки
- `DealMessage` - сообщения
- `DealDispute` - споры
- `Transaction` - транзакции
- `SellerWithdrawal` - выводы
- `SellerReview` - отзывы
- `Favorite` - избранное
- `Notification` - уведомления

### Сервисы:
- `SellerService` - управление продавцами
- `LotService` - управление лотами
- `DealService` - управление сделками

### Хендлеры:
- `seller.py` - функции продавца
- `deals.py` - управление сделками

## 🎬 Примеры использования

### Покупка с автовыдачей:
```
1. Выбрать товар
2. Выбрать лот (по цене/рейтингу)
3. Купить
4. Получить код мгновенно
5. Подтвердить
```

### Создание лота:
```
1. /seller
2. Добавить лот
3. Загрузить коды
4. Активировать
5. Готово!
```

### Ручная доставка:
```
1. Покупатель покупает
2. Открывается чат
3. Продавец отправляет товар
4. Покупатель подтверждает
5. Средства переводятся
```
