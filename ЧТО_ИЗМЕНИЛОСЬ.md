## 🚀 Как запустить

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Запустить бота (применит миграции)
```bash
python -m app.main
```

### 3. Запустить API (в отдельном терминале)
```bash
python -m api.main
```

### 4. Открыть документацию
```
http://localhost:8000/api/docs
```

## 📞 Поддержка

Если что-то не работает:

1. **Бот не запускается**
   - Проверь BOT_TOKEN в .env
   - Проверь DATABASE_URL
   - Проверь что PostgreSQL запущен

2. **API не запускается**
   - Проверь что порт 8000 свободен
   - Проверь что установлен FastAPI
   - Проверь Python версию (нужен 3.12+)

3. **Ошибки миграций**
   - Удали все таблицы и запусти бота заново
   - Проверь версию PostgreSQL (нужен 15+)

4. **Import errors**
   - Переустанови зависимости: `pip install -r requirements.txt`
