## 🚀 Как запустить

### Backend:
```bash
cd ПРОЕКТЫ/gpay-main
uvicorn api.main:app --reload
```

### Frontend:
```bash
cd ПРОЕКТЫ/gpay-main/frontend
npm install  # если еще не установлены зависимости
npm run dev
```

Откроется на http://localhost:3000
