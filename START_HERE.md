## 📋 Шаг 1: Настрой .env файл

Открой файл `.env` в корне проекта и заполни:

```env
# Токен твоего бота от @BotFather
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz

# Данные PostgreSQL (если на bothost.ru - узнай в панели)
DATABASE_URL=postgresql+asyncpg://user:password@localhost:5432/game_pay

# Redis (обычно стандартный)
REDIS_URL=redis://localhost:6379/0

# Твой Telegram ID (узнай у @userinfobot)
ADMIN_IDS=123456789
```

## 📋 Шаг 6: Протестируй основные функции

### Тест 1: Регистрация продавца
1. Открой бота в Telegram
2. Отправь `/seller`
3. Введи название магазина (например: "My Shop")
4. Должно появиться: "Магазин создан! Ожидает модерации"

### Тест 2: Активируй продавца (через SQL)
```sql
-- Подключись к БД
psql -d game_pay

-- Найди своего продавца
SELECT * FROM sellers WHERE user_id IN (
    SELECT id FROM users WHERE telegram_id = ВАШ_TELEGRAM_ID
);

-- Активируй
UPDATE sellers SET status = 'active' WHERE id = ID_ПРОДАВЦА;
```

### Тест 3: Создай лот
1. Отправь `/seller` снова
2. Выбери "➕ Добавить лот"
3. Следуй инструкциям
4. Добавь коды (если автовыдача)
5. Активируй лот

### Тест 4: Проверь каталог
1. Отправь `/start`
2. Выбери "🎮 Игры"
3. Выбери игру
4. Выбери категорию
5. Выбери товар
6. Должны появиться лоты от разных продавцов!

### Тест 5: Купи товар (если есть баланс)
```sql
-- Пополни баланс для теста
UPDATE users SET balance = 1000 WHERE telegram_id = ВАШ_TELEGRAM_ID;
```

1. Выбери лот
2. Нажми "🛒 Купить"
3. Подтверди покупку
4. Если автовыдача - получишь код сразу
5. Если ручная - откроется чат

### Тест 6: Проверь сделки
1. Отправь `/deals`
2. Должна появиться твоя сделка
3. Открой детали
4. Проверь чат (если ручная доставка)

## 🆘 Если что-то не работает

### Ошибка миграции:
```bash
# Проверь текущую версию
alembic current

# Если нужно, откати
alembic downgrade -1

# Попробуй снова
alembic upgrade head
```

### Ошибка импорта:
```bash
pip install -r requirements.txt --upgrade
```

### Бот не запускается:
```bash
# Проверь логи
python -m app.main

# Или запусти с отладкой
python -m app.main --debug
```

### Таблицы не созданы:
```bash
# Проверь в PostgreSQL
psql -d game_pay -c "\dt"

# Должны быть: sellers, lots, deals, transactions и т.д.
```
