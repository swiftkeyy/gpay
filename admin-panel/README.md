# GamePay Admin Panel

Простая веб-админка для управления P2P маркетплейсом GamePay.

## Возможности

- **Dashboard**: Статистика по пользователям, продавцам, лотам, сделкам, спорам и выводам
- **Users**: Просмотр пользователей, блокировка/разблокировка
- **Sellers**: Управление продавцами (одобрение/отклонение заявок)
- **Lots**: Управление лотами (одобрение/отклонение)
- **Disputes**: Разрешение споров (возврат покупателю / выплата продавцу)
- **Withdrawals**: Обработка выводов средств (одобрение/отклонение)

## Как использовать

### Локально

Просто откройте `index.html` в браузере.

### На хостинге

1. Загрузите файл на любой статический хостинг (Vercel, Netlify, GitHub Pages)
2. Или разместите в папке `static` на Railway и настройте nginx для раздачи

### Настройка

В файле `index.html` измените:

```javascript
const API_URL = 'https://gpay-production-5cd1.up.railway.app/api/v1';
const USER_ID = 1; // ID админа
```

## Безопасность

⚠️ **ВАЖНО**: Эта админка не имеет встроенной аутентификации. Для production:

1. Добавьте авторизацию через Telegram Web App
2. Или используйте HTTP Basic Auth на уровне nginx
3. Или добавьте JWT токены

## Технологии

- HTML + Tailwind CSS
- Vanilla JavaScript
- Fetch API для работы с backend
