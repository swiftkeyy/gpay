# Playerok-Style Improvements ✨

## Что сделано

### 1. Реальные изображения игр 🎮
- Добавлен маппинг `GAME_IMAGES` с реальными URL изображений для популярных игр:
  - Brawl Stars
  - Roblox
  - Genshin Impact
  - CS2
  - Minecraft
  - Standoff 2
  - Fortnite
  - Valorant
  - PUBG
  - Mobile Legends

### 2. API улучшения 🚀

#### `/api/v1/games`
- Теперь возвращает `image_url` для каждой игры
- Использует реальные изображения из `GAME_IMAGES`

#### `/api/v1/lots`
- Добавлены поля:
  - `game_slug` - slug игры для идентификации
  - `game_name` - название игры
  - `game_image_url` - URL изображения игры
- Изображения товаров теперь основаны на игре (первое фото = фото игры)
- `delivery_type` теперь корректно определяется из `fulfillment_type`

#### `/api/v1/lots/{id}`
- Добавлены поля:
  - `game_slug`
  - `game_name`
  - `game_image_url`
  - `category_name`
  - `seller_total_sales`
  - `total_reviews`
  - `delivery_time_minutes`
- Полная информация о продавце и товаре

#### `/api/v1/cart`
- Изображения товаров в корзине теперь используют реальные фото игр
- `delivery_type` корректно определяется

#### `/api/v1/orders`
- Изображения товаров в заказах используют реальные фото игр

### 3. Frontend улучшения (Playerok-style) 💎

#### Главная страница (HomePage)
- **Горизонтальные категории игр** - как в Playerok:
  - Скроллируемая полоса с кнопками игр
  - Активная игра подсвечивается фиолетовым
  - Кнопка "Все игры" для сброса фильтра
  
- **Улучшенные карточки товаров**:
  - Градиентный фон (purple-blue)
  - Бейдж "⚡ Авто" для авто-выдачи (зелёный градиент)
  - Бейдж "⭐ TOP" для featured товаров (жёлто-оранжевый градиент)
  - Цена крупным фиолетовым шрифтом
  - Рейтинг жёлтыми звёздами
  - Информация о продавце с рейтингом
  - Кнопка "В корзину" с градиентом (purple-blue)
  
- **Улучшенный поиск и фильтры**:
  - Закруглённые инпуты (rounded-xl)
  - Focus ring фиолетового цвета
  - Активные фильтры подсвечиваются
  - Чистый белый фон с тенями

#### Страница товара (ProductPage)
- Добавлены поля:
  - `game_name` - отображается как тег
  - `category_name` - отображается как тег
  - `seller_total_sales` - количество продаж продавца
  - `total_reviews` - общее количество отзывов
  - `delivery_time_minutes` - время доставки для авто-выдачи

### 4. Цветовая схема 🎨
- Основной цвет: фиолетовый (`purple-600`)
- Акцентный цвет: синий (`blue-600`)
- Градиенты: `from-purple-600 to-blue-600`
- Бейджи с градиентами для визуальной привлекательности
- Тени для глубины (`shadow-sm`, `shadow-md`, `shadow-lg`)
