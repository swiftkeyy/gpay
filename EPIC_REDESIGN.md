# 🎮 GAME PAY - EPIC CYBERPUNK GAMING REDESIGN

## 🔥 ЧТО СДЕЛАНО

Полный редизайн маркетплейса в стиле **Cyberpunk 2077 + Gaming HUD** с неоновыми эффектами, glassmorphism и премиум-анимациями.

---

## 🎨 ДИЗАЙН-СИСТЕМА

### Цветовая палитра
```css
Основной фон: #0A0A0F (cyber-dark) - глубокий чёрный
Неоновый красный: #FF0033 (neon-red) - основной акцент
Неоновый циан: #00F0FF (neon-cyan) - вторичный акцент
Неоновый фиолетовый: #9D4EDD (neon-purple) - третичный акцент
Неоновое золото: #FFD700 (neon-gold) - премиум элементы
```

### Шрифты
- **Orbitron** - основной gaming шрифт (жирный, футуристичный)
- **Rajdhani** - дополнительный шрифт (лёгкий, читаемый)

## ✨ КЛЮЧЕВЫЕ ФИЧИ

### 1. Эпичная шапка с логотипом
- **Логотип Game Pay** с неоновым свечением
  - Буквы "GP" с красным glow-эффектом
  - Текст "GAME PAY" с киберпанк-стилем
  - Подпись "PREMIUM MARKETPLACE"
  - Анимированная неоновая линия снизу

- **Баланс пользователя**
  - Иконка алмаза 💎
  - Glassmorphism фон
  - Золотая рамка с свечением

### 2. Поиск с неоновым эффектом
- Glassmorphism input с blur-эффектом
- Неоновая красная рамка
- Hover: усиление свечения + scale
- Focus: пульсирующий glow
- Placeholder: "🔍 Search epic items..."

### 3. Категории игр - Cyberpunk Style
- Кнопки в стиле gaming chips
- Активная кнопка:
  - Градиент red → purple
  - Пульсирующее свечение
  - Анимация pulse-glow
- Неактивные кнопки:
  - Полупрозрачный чёрный фон
  - Серая рамка
  - Hover: неоновая рамка

### 4. Фильтры и сортировка - Gaming HUD
- Кнопка фильтров с иконкой 🎯
- Dropdown в стиле киберпанк
- Опции:
  - 🔥 POPULAR
  - 💰 CHEAP FIRST
  - 💎 EXPENSIVE FIRST
  - ⚡ NEWEST
  - ⭐ TOP RATED

### 5. Карточки товаров - Glassmorphism
**Структура карточки:**
```
┌─────────────────────────────┐
│  [Изображение игры]         │ ← Hover: scale 110%
│  [Game Badge]    [EPIC]     │ ← Бейджи с неоном
│  [⚡ INSTANT]               │
│  └─ Gradient Overlay        │
├─────────────────────────────┤
│  Название товара            │ ← Hover: cyan glow
│  999₽        ⭐ 5.0        │ ← Красная цена с glow
│  Seller • ⭐ 5.0           │
│  [🛒 BUY NOW]              │ ← Градиент кнопка
└─────────────────────────────┘
```

**Эффекты:**
- Glassmorphism фон (black/40 + blur)
- Неоновая красная рамка (hover: усиление)
- Hover: scale 102% + shadow-neon-red-lg
- Изображение: zoom 110% при hover
- Gradient overlay снизу

**Бейджи:**
- **Game Badge** (top-left): циан, glassmorphism
- **EPIC Badge** (top-right): золотой градиент, pulse-glow
- **INSTANT Badge** (bottom-left): зелёный градиент

### 6. Нижняя навигация - Gaming HUD
**Стиль:**
- Backdrop blur + cyber-dark фон
- Неоновая красная рамка сверху
- Анимированная линия (pulse-glow)

**Иконки:**
- 🏠 HOME (красный при активном)
- 🛒 CART (с счётчиком в неоновом badge)
- 📦 ORDERS
- 💼 SELL (золотой для продавцов)
- 👤 PROFILE (фиолетовый)

**Эффекты:**
- Активная иконка: scale 110% + float animation + glow
- Неактивная: серая, hover → cyan
- Счётчик корзины: градиент red→purple, pulse-glow

## 🎬 АНИМАЦИИ

### Pulse Glow
```css
Пульсирующее неоновое свечение
Используется: активные кнопки, бейджи, счётчики
```

### Float
```css
Плавающая анимация вверх-вниз
Используется: активные иконки навигации
```

### Scale on Hover
```css
Увеличение при наведении (102-110%)
Используется: карточки, кнопки
```

### Gradient Shift
```css
Смена градиента при hover
from-neon-red → from-neon-purple
```
